DSA Visualizer — Interactive Data Structure Learning Platform

An interactive visualization platform for learning Data Structures & Algorithms step-by-step.
Built using HTML, CSS, JavaScript, with C++ implementations for algorithm reference.

This project helps students understand DSA visually instead of only reading theory.

🌐 Live Demo
Run index.html using Live Server
📌 Features

✅ Array Search Visualization (Linear & Binary)
✅ Duplicate Detection
✅ Expression Converter (Infix → Postfix / Prefix)
✅ Graph Traversal (BFS & DFS)
✅ LRU Cache Simulator
✅ OOP Concepts (C++)
✅ Dark Mode UI
✅ Step-by-Step Animation
✅ Interactive Controls
✅ Modern Glass UI
